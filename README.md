# BlackSheep Culture

Rebellious streetwear for the resilient. Defy the flock. Live limitless.

## Deployment

This is a static site. To deploy with HTTPS:

1. Host on [Netlify](https://netlify.com) or [Vercel](https://vercel.com) for free HTTPS.
2. Upload the files or connect your Git repo.
3. Update the canonical URLs in `index.html`, `robots.txt`, and `sitemap.xml` to match your domain.

## Images & Video

Replace placeholder media in the `images/` and `video/` folders:

- `images/hero.jpg`: Hero background
- `images/product1.jpg`, `images/product2.jpg`, `images/product3.jpg`: Product images
- `images/lookbook1.jpg`, `images/lookbook2.jpg`, `images/lookbook3.jpg`: Lookbook gallery
- `images/og-image.jpg`: Social media sharing image
- `video/brand-film.mp4`: Brand film (used in the Brand Film section)

See `images/README.md` and `video/README.md` for details.

## SEO & Security

- Basic meta tags and canonical URL included.
- Robots.txt and sitemap.xml provided.
- CSP for security.
