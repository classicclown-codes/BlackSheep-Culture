# Images Folder

Place your images here to replace the placeholders on the site.

- `logo.png`: Brand logo (square, 64x64 recommended, PNG with transparency)
- `hero.jpg`: Hero section image (900x600 recommended)
- `product1.jpg`: First product image (e.g., hoodie)
- `product2.jpg`: Second product image (e.g., pants)
- `product3.jpg`: Third product image (e.g., tee)
- `lookbook1.jpg`: First lookbook image
- `lookbook2.jpg`: Second lookbook image
- `lookbook3.jpg`: Third lookbook image

All images should be optimized for web (JPEG/PNG/WebP, <500KB each).
