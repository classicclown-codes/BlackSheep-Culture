# Video Folder

Place your brand films here:

- `brand-film.mp4` (default) 
- `brand-film-2.mp4`
- `brand-film-3.mp4`

The site renders a player that can switch between videos via the playlist in the Brand Film section.

**Reminder:** Keep files reasonable (< 15MB each) and encode as H.264 or H.265 for broad compatibility.
